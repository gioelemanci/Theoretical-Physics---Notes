\section{Overview}
We begin by testing some custom macros:
\[
  \dvf{f(x)}{x} = \frac{\dd f}{\dd x}, \qquad
  \pdvT{p}{v} = \left(\frac{\partial p}{\partial v}\right)_T
\]

Also note that we can use physical quantities:
\begin{equation}
  E = \qty{3.5e-19}{\joule} = \qty{2.18}{\electronvolt}.
\end{equation}

\begin{remark}
  The use of \verb|\qty| requires the \texttt{siunitx} package.
\end{remark}
